🛒 Walmart Sales Dashboard | Power BI
📌 Project Title:-
Walmart Sales Dashboard using Power BI  # Walmart-Sales-Dashboard

1.Project Description
This Power BI dashboard analyzes Walmart's weekly sales performance using historical sales and economic indicator data. The dashboard provides interactive visualizations to help users understand sales trends, the impact of holidays, and the relationship between weekly sales and external factors such as fuel price, temperature, CPI, and unemployment.
The project demonstrates data cleaning, data modeling, DAX calculations, and interactive dashboard design to support business decision-making.

2.Source: Kaggle – Walmart Sales Dataset
Description:
The dataset was downloaded from Kaggle and contains historical Walmart weekly sales data along with economic indicators such as Fuel Price, CPI, Temperature, Unemployment, Holiday Flag, and Date. It is used for practicing data analysis, visualization, and dashboard development.

3.Project Purpose
The objective of this project is to:
Analyze weekly sales performance.
Monitor sales trends over time.
Compare holiday and non-holiday sales.
Study the relationship between sales and economic factors.
Build an interactive dashboard for business insights.

4.Tech Stack
Power BI Desktop
Power Query
DAX (Data Analysis Expressions)
Data Modeling
Microsoft Excel / CSV Dataset
Data Visualization

4. 📊 Dashboard Features
KPI Cards
Total Sales
Average Weekly Sales
Holiday Sales
Maximum Weekly Sales
Minimum Weekly Sales

5.Project Highlights
Clean and interactive dashboard design.
Dynamic filtering using slicers.
DAX measures for KPI calculations.
Time-based sales analysis using a Calendar table.
